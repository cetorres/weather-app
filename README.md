# Weather app

A weather app using Open Meteo (https://open-meteo.com/) API.

## Getting Started

### Android APK

Just copy and install the [weather-app-release.apk](weather-app-release.apk) on your Android device.

### Debug run

Run:

```sh
$ flutter run
```

And select your device or emulator.
